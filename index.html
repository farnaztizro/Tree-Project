As per the requirements of this course, we will be using a dataset from Kaggle and will analyze that dataset by visualizing its characteristics using different parameters. The chosen dataset is the 5M Trees dataset available here

Design Choice For Graph 1:

Since our data is of categorical data type, so we chose horizontal bar graph to represent it as it is easier to display it. Also, our dataset is relatively big, horizontal bar graph tends to work better in narrow layouts

Design Choice For Graph 5(heatmap):

For this part we are required to create heatmap for different cities. Keeping our dataset in view, we had to preprocess the data in such a way that we need to keep the common trees for all selected cities. This approach allows us to visualize the results more effectively. Plus since there are big abnormalities in the count of trees, we chose the log scale for displaying colors.
